```python
# State variables:
self._cursor: int = -1

# BINDINGS:
BINDINGS = [
    Binding("escape", "focus_projects", "Back"),
    Binding("up", "cursor_up", "Up"),
    Binding("down", "cursor_down", "Down"),
    Binding("k", "cursor_up", "Up"),
    Binding("j", "cursor_down", "Down"),
    Binding("enter", "focus_session", "Open"),
]

def _update_highlight(self) -> None:
    for row in self._rows:
        row.remove_class("--highlight")
    if 0 <= self._cursor < len(self._rows):
        self._rows[self._cursor].add_class("--highlight")
        self._rows[self._cursor].scroll_visible()

def action_cursor_up(self) -> None:
    if self._cursor > 0:
        self._cursor -= 1
        self._update_highlight()

def action_cursor_down(self) -> None:
    if self._cursor < len(self._rows) - 1:
        self._cursor += 1
        self._update_highlight()

def action_focus_projects(self) -> None:
    self.app.query_one("#project-list").focus()
```

| WorktreePane → webbrowser | URL from MRInfo.url opened in system browser |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-k2u-01 | Tampering | subprocess.run path arg | accept | WorktreeInfo.path comes from `git worktree list` output (trusted local process). Not user-supplied at runtime. Risk minimal for a personal macOS tool. |
| T-k2u-02 | Elevation | config.ide open -a | accept | ide field is from ~/.joy config — user-controlled machine config. No privilege boundary crossed. |
| T-k2u-03 | Information | webbrowser.open MR URL | accept | URL originates from GitHub/GitLab API response (forge-trusted). Opens in user's default browser — no data exfiltration risk. |
</threat_model>
