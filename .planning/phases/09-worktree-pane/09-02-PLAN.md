**3. Pure functions for path processing (D-13, D-14):**

`abbreviate_home(path_str: str) -> str`:
- Get `home = str(Path.home())`
- If `path_str.startswith(home)`: return `"~" + path_str[len(home):]`
- Otherwise return `path_str` unchanged
- Edge case: if path_str == home exactly, returns "~"

`middle_truncate(path: str, max_width: int) -> str`:
- If `len(path) <= max_width`: return path unchanged
- Split path on `"/"`
- If `len(parts) <= 3`: fall back to right-truncation: `path[:max_width - 1] + "\u2026"`
- Otherwise: `head = parts[0] + "/" + parts[1]` (e.g. "~/Github"), `tail = parts[-1]`, `ellipsis = "/\u2026/"`
- If `len(head) + len(ellipsis) + len(tail) > max_width`: fall back to right-truncation
- Otherwise: return `head + ellipsis + tail`
