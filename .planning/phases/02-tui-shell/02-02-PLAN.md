   - The renderable is computed by a method `_render_text(self) -> str` which returns: `f"{icon}  {label}  {value}"` where:

Since Textual Static widgets render Rich text and Textual CSS supports `overflow: hidden`, use CSS on the row. The ObjectRow should have a CSS class or inline style. Add to the widget:
   ```python
   DEFAULT_CSS = """
   ObjectRow {
       width: 1fr;
       height: 1;
       padding: 0 1;
       overflow: hidden;
   }
   """
   ```
   The single-line height (height: 1) combined with overflow: hidden ensures values are truncated at the widget boundary.
