Add a new public method to `WorktreePane`:

```python
def set_refresh_label(self, timestamp: str, *, stale: bool = False) -> None:
    """Update border_title with refresh timestamp. Stale adds warning icon.

    Args:
        timestamp: Human-readable time string (e.g., "2m ago", "14:32").
        stale: If True, prefix timestamp with warning icon.
    """
    if stale:
        self.border_title = f"Worktrees  \u26a0 {timestamp}"
    else:
        self.border_title = f"Worktrees  {timestamp}"
```

Note: Two spaces between "Worktrees" and the timestamp/icon for visual breathing room. The `\u26a0` is the standard Unicode warning sign. The stale color (yellow/orange) will be handled via Rich markup in the border_title if Textual supports it — but since `border_title` is a plain string in Textual (not Rich Text), the color will be handled by the border CSS class. If Textual's `border_title` does not support Rich markup, accept plain text with the icon as the stale indicator. Do NOT attempt `border_title = Text(...)` — Textual expects a string.
