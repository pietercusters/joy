    In `src/joy/widgets/object_row.py`, append the following CSS rules inside the `DEFAULT_CSS` string of the `ObjectRow` class, after the existing `.col-kind` rule:

    ```css
    ObjectRow.--stale .col-value {
        text-style: italic;
        color: $text-muted;
    }
    ObjectRow.--stale .col-icon {
        color: $text-muted;
    }
    ObjectRow.--stale .col-kind {
        text-style: italic;
        color: $text-muted;
    }
    ```

    This dims all three columns and italicizes value and kind columns when the row has the `--stale` CSS class. The `$text-muted` color is already used by `.col-kind` normally, so stale styling extends it to icon and value columns too.

    **Step 2: Apply --stale class during _render_project (D-08)**

    In `src/joy/widgets/project_detail.py`, in the `_render_project` method, after the line `row = ObjectRow(item, index=row_index)` and before `scroll.mount(row)`, add:

    ```python

        row.add_class("--stale")
    ```

    The `getattr` with default False provides safety against ObjectItems that somehow lack the stale attribute (backward compatibility). The full block becomes:

    ```python
    for item in group_items:

            row.add_class("--stale")
    ```

    **Step 3: Add integration test for stale CSS class**

    In `tests/test_propagation.py`, add a test class `TestStaleCSSIntegration`:

    ```python
    class TestStaleCSSIntegration:
        """Test that stale ObjectItems get --stale CSS class on ObjectRow."""

        def test_stale_row_has_css_class(self):
            """ObjectRow created with stale=True item gets --stale class applied in _render_project."""
            item = ObjectItem(kind=PresetKind.AGENTS, value="claude-work", stale=True)
            row = ObjectRow(item, index=0)
            # Simulate what _render_project does
            if getattr(item, 'stale', False):
                row.add_class("--stale")
            assert row.has_class("--stale")

        def test_non_stale_row_no_css_class(self):
            """ObjectRow created with stale=False item does NOT get --stale class."""
            item = ObjectItem(kind=PresetKind.AGENTS, value="claude-work", stale=False)
            row = ObjectRow(item, index=0)
            if getattr(item, 'stale', False):
                row.add_class("--stale")
            assert not row.has_class("--stale")
    ```

    Note: `has_class` and `add_class` are Textual widget methods that work without a running app for basic class tracking. If `has_class` requires a mounted widget, wrap the test in an async app test harness (similar to test_refresh.py pattern).
